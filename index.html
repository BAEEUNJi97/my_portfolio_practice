About

패션 디자이너로 일하다 직접 화면을 만들고 기능이 동작하는 순간에 빠져들었습니다. 디자인 현장에서 쌓은 시각적 감각과 사용자 중심 사고를 코드에 그대로 가져왔습니다.

React와 Next.js 기반 프로젝트를 통해 UI 구현과 데이터 처리 흐름을 경험하면서, 기능 완성 이후에도 수정과 보완이 반복된다는 걸 자연스럽게 체감했습니다. 그래서 처음부터 화면 단위 역할을 명확히 나누고, 변경에 강한 구조를 우선하게 되었습니다.

아직은 신입이지만 실무에 빠르게 녹아들 수 있도록 코드잇 프론트엔드 심화 과정을 마치며 실전 역량을 계속 키우고 있습니다.
