# 포트폴리오 프로젝트 구조

## 📁 프로젝트 구조

```
real_eunji/
├── assets/                 # 정적 자산
│   ├── css/               # 스타일시트
│   │   ├── reset.css      # CSS 리셋
│   │   ├── variables.css  # CSS 변수
│   │   ├── components.css # 컴포넌트 스타일
│   │   └── style.css      # 메인 스타일
│   ├── js/                # JavaScript 파일
│   │   ├── utils.js       # 유틸리티 함수
│   │   ├── navigation.js  # 네비게이션 관련
│   │   ├── animations.js  # 애니메이션 관련
│   │   └── script.js      # 메인 스크립트
│   ├── images/            # 이미지 파일
│   ├── fonts/             # 폰트 파일
│   └── icons/             # 아이콘 파일
├── docs/                  # 문서
│   └── README.md          # 프로젝트 문서
├── index.html             # 메인 HTML 파일
├── package.json           # 프로젝트 설정
├── .gitignore            # Git 무시 파일
└── README.md             # 프로젝트 설명
```

## 🚀 시작하기

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

## 📝 주요 기능

- **반응형 디자인**: 모든 디바이스에서 최적화된 경험
- **모듈화된 코드**: 유지보수가 쉬운 구조
- **접근성**: 웹 접근성 가이드라인 준수
- **성능 최적화**: 빠른 로딩과 부드러운 애니메이션

## 🛠 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 모던 CSS 기능 활용
- **JavaScript ES6+**: 모듈 시스템 사용
- **반응형 디자인**: 모바일 퍼스트 접근
