# Portfolio Website

연하늘색과 연그레이를 조합한 모던한 포트폴리오 웹사이트입니다.

## 🎨 디자인 특징

- **색상 팔레트**: 연하늘색(#38bdf8, #0ea5e9, #3b82f6)과 연그레이 조합
- **글래스모피즘 디자인**: 반투명 유리 효과와 블러 처리
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기 지원
- **부드러운 애니메이션**: CSS3와 JavaScript를 활용한 인터랙티브 효과

## 📁 프로젝트 구조

```
real_eunji/
├── index.html              # 메인 HTML 파일
├── README.md              # 프로젝트 설명서
├── public/                # 정적 파일 (이미지, 아이콘 등)
└── src/                   # 소스 코드
    ├── styles/            # CSS 스타일시트
    │   └── style.css      # 메인 스타일 파일
    ├── scripts/           # JavaScript 파일
    │   └── script.js      # 메인 스크립트 파일
    └── assets/            # 기타 리소스
        └── images/        # 이미지 파일들
```

## 🚀 시작하기

1. 프로젝트 클론 또는 다운로드
2. `index.html` 파일을 웹 브라우저에서 열기
3. 로컬 서버 실행 (선택사항):

   ```bash
   # Python 3 사용시
   python -m http.server 8000

   # Node.js 사용시
   npx serve .
   ```

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, 애니메이션, 글래스모피즘
- **JavaScript (ES6+)**: DOM 조작, 이벤트 처리, 애니메이션
- **Google Fonts**: Noto Sans KR 폰트

## 📱 반응형 지원

- **데스크톱**: 1200px 이상
- **태블릿**: 768px - 1199px
- **모바일**: 767px 이하

## 🎯 주요 기능

- 부드러운 스크롤 네비게이션
- 인터랙티브 호버 효과
- 파티클 애니메이션
- 타이핑 효과
- 리플 효과
- 스크롤 기반 애니메이션

## 📝 커스터마이징

### 색상 변경

`src/styles/style.css` 파일에서 CSS 변수를 수정하여 색상을 변경할 수 있습니다.

### 콘텐츠 수정

`index.html` 파일에서 텍스트와 정보를 수정할 수 있습니다.

### 애니메이션 조정

`src/scripts/script.js` 파일에서 JavaScript 애니메이션을 조정할 수 있습니다.

## 📄 라이선스

이 프로젝트는 개인 포트폴리오 용도로 제작되었습니다.
